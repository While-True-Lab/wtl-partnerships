# Partnership Templates

## Available Templates

| Template | Use Case |
|----------|----------|
| [NDA.md](NDA.md) | Mutual non-disclosure agreement |
| [PKS.md](PKS.md) | Partnership agreement (Perjanjian Kerja Sama) |

## How to Use

1. Copy template to deal folder
2. Fill in bracketed placeholders
3. Review with Aldo for commercial terms
4. Send to partner for review
5. Legal review for deals > IDR 500M
6. Sign and file in `deals/[deal-name]/contracts/`

## Customization Notes

- Always update entity names (PT While True Lab vs product entities)
- Adjust revenue share based on partnership type
- Territory can be exclusive or non-exclusive
- Governing law: Indonesia (BANI arbitration)
